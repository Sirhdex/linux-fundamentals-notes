## Concepts
- Linux accesses / manages files through terminal, Windows through GUI
- Terminals run commands that allow you to modify files selectively using Regex + able to run multiple commands

## Commands
- touch-- create files
- mkdir-- create directories
- mkdir -p-- making directories in dirctories (spawn parent directories)
- tree-- displays whole structure (file structure)
- tree [.]-- displays structure from current directory
- mv-- move / rename file
- cp-- copy files

## Examples
- touch ./etc/storage/new_text.txt
- mkdir -p /storage/local/documents/new_dir
- mv old_text.txt new_text.txt
- mv old_text.txt new_text.txt /storage
