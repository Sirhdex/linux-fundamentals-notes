## Concepts
- Navigation essential for traversing between different directories and files


## Commands Learned
- pwd-- tells current directory
- ls-- lists files in directory
- ls [directory]-- lists files without being in directory
- ls -l-- lists files + more information
- ls -la-- lists all files
- ls -i-- lists files with indoe number
- cd-- change directories
- cd [-]-- change to previous directory
- cd [..]-- change to parent directory (back one)
- &&-- conditional (only execute B if A is true)
- |-- piping (take output from A and pipe it to B)
- env-- prints environment (displays all name value pairs of session)
- ifconfig-- network interface information / interfaces

- cd etc && clear
- ls -la etc | grep 'bash'
