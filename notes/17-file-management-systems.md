## Concepts
- Linux is a verstile OS that supports many different file systems
- Each file system is made towards a specifc purpose
- These structures consists of several components, most importantly Inodes; inodes store metadata about each file and directory
- Inode table is a collection of inodes, essentially a database for tracking every file and directory

## Commands
- fdisk -- main tool for disk management; allowing for creation, deletion, or managing of partisions
- mount -- manually mount file systems
- umount -- unmount file system

## Examples
- fdisk -l -- lists disks paritions
