## Concepts
- Network Services are designed to perform specific task; many of which involving remote connection
- Open SSH is a free and open-source implementation of SSH
- Admins use Open SSH to manage remote systems by establishing an encrypted connection to a remote host
- NFS is a network protocol that allows us to store and manage remote systems as if they were on the local system
- A web server is software that delivers data, documents, applications, and various functions over the Internet; HTTP used transmit data and recieve requests

## Commands
- ssh -- network protocol that allows for secure transmission of data and commands over the internet
- python3 -- starts python web server
- openvpn -- connects to openvpn server

## Examples
- ssh [hostname]@[IP_Address] -- connects to remote host
- python3 -m http.server 443 -- creates a python web server listening a port 443
- openvpn [*.ovpn]
