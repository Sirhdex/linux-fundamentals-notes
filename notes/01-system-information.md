## Concepts
- Important to understand all of the fundamentals of Linux systems
- Being familiar with key tools helps understanding

## Commands
- man [command]-- gives manual of command
- [command] --help-- helps with information on how to use command
- whoami-- displays current username
- id-- returns user id
- hostname-- print the name of computer 
- uname-- print system information
- uname -r-- system information (print kernel release)
- uname -i-- system information (print hardware platform)

## Examples
- man whoami
- whoami --help
